This folder ocntians subfolders with supplementary data for the paper: 
"Exploring Carbon Source Related Localization and Phosphorylation in the Snf1/Mig1 Network using Population and Single Cell-based Approaches"

it contains the following subfolders: 
Monolix_input: processed and simplified FRAP data for use in monolix 2023R1
Monolix-output: resultfiles generated during run of monolix and as presented in the paper

Processed-data_FRAP: all data used for modelling in monolix including calculations and more thorough description 
Rawdata_FRAP: all rawdata acquired during FRAP experiments, including data sorted out due to cell movement
Movies_FRAP: exemplary movies of the FRAp experiments for one cell per condition
